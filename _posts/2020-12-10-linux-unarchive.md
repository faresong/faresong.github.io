## 0. 快速使用

```bash
tar xvf demo.tar
tar zxvf demo.tar.gz
tar jxvf demo.tar.bz2
tar Jxvf demo.tar.xz
unzip demo.zip
7za x demo.7z
```

## 1. tar

系统一般自带 tar 打包工具。可协同 gzip (gz), bzip2 (bz2), lzma (xz) 等压缩工具。

**tar**

## 2. zip

`apt install unzip`

## 3. 7z

`apt install p7zip`
